# Factigent Design

A simple static landing page built with HTML and CSS. The page features a modern two-panel layout with a branded hero section and a polished authentication form.

## Files

- [factigent-design.html](factigent-design.html) — the complete single-file implementation of the original design
- [index.html](index.html) — a landing page that links to the new theme gallery
- [theme-suggestions.html](theme-suggestions.html) — a themed showcase with Ocean, Sunset, and Forest palettes

## Preview locally

Open [factigent-design.html](factigent-design.html) in your browser to view the original page.

If you prefer a local web server, you can run:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000/.

## Notes

This project is intentionally lightweight and contains no external dependencies.
