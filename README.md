# Factigent Design

A simple static landing page built with HTML and CSS. The page features a modern two-panel layout with a branded hero section and a polished authentication form.

## Files

- [factigent-design.html](factigent-design.html) — the complete single-file implementation of the design

## Preview locally

Open [factigent-design.html](factigent-design.html) in your browser to view the page.

If you prefer a local web server, you can run:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000/factigent-design.html.

## Notes

This project is intentionally lightweight and contains no external dependencies.
